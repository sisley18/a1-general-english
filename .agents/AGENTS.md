# Project Rules

## UI/UX Layout Model for all courses
When creating or modifying course materials and web interfaces, you MUST use the following layout model requested by the user:
- **Top Navigation Bar**: Floating, pill-shaped, using light/warm colors. No left sidebar.
- **Main Container**: A large translucent glassmorphic container (frosted glass over a warm glowing background) to hold the content.
- **Two-Column Grid Layout for units**:
  - **Left Column (approx 60%)**: A large embedded YouTube video player with a warm glowing border/shadow.
  - **Right Column (approx 40%)**: "Vocabulary Boost" cards stacked vertically next to the video.
- **Vocabulary Cards Design**: White background, rounded corners, soft shadow. Should feature an icon on the left, the vocabulary word/definition in the center, and a bright gradient "Play" (audio) button on the right.
- **Color Palette**: Strictly warm/light colors (soft peaches, warm whites, gentle yellows, soft warm greys). Absolutely NO dark mode or black backgrounds.
